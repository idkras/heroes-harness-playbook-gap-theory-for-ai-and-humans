# Template: Owner Effort Digest

> Финальный блок substantial delivery (mandatory section #10). Прозрачно показывает владельцу: что осталось ему сделать, насколько это тяжело по шкале 0-100, и что агент может взять на себя.

## Шаблон

### Если есть шаги для человека

```
## Owner effort digest

| # | Next action | Усилие человека (0-100) | Что агент может сделать сам |
|---|---|---|---|
| 1 | <конкретное действие 1 фразой> | <балл по 02-rca-effort-scale-0-100.md> | <что я уже сделал / могу сделать без тебя> |
| 2 | ... | ... | ... |
| 3 | ... | ... | ... |

Total owner effort: <сумма> единиц
Critical path: <что блокирует closure если не сделано>
```

### Если нет шагов для человека

```
## Owner effort digest

Нет шагов для человека.

Причина: <одна фраза>
- "только информация — действий не требуется"
- "доставка завершена end-to-end, агент сделал всё сам"
- "следующий шаг автоматический (cron / workflow / скилл)"

Что я (агент) сделаю дальше без твоего вмешательства:
- <автоматическое действие 1>
- <автоматическое действие 2>
```

## Правила содержания

| Правило | Что значит |
|---|---|
| Любой клик владельца = минимум 50 | Открытие документа, переключение в другой инструмент, чтение > 1 экрана |
| Просьба «выбери вариант» = 20 | Если варианты предложены и сравнены |
| «Подтверди и я продолжу» = 10 | Только yes/no |
| «Прочитай результат на 5 страниц и одобри» = 50-70 | Зависит от плотности материала |
| «Перепиши вручную» = 90-100 | Это переделка, не правка |
| **+50 надбавка** за легенду / отдельный мини-словарь | Голые коды без расшифровки в той же строке |
| **+100 надбавка** за провал доверия | Доставка с обманчивыми claims, разрыв обещаний |

## Worked example

### Хороший пример

```
| # | Next action | Усилие человека (0-100) | Что агент может сделать сам |
|---|---|---|---|
| 1 | Подтвердить выбор лицензии (A — Apache+CC BY 4.0 / B — Apache+CC BY-NC) | 20 (выбор из 2 предложенных вариантов) | Я уже подготовил оба варианта rationale; могу сразу ставить твой выбор |
| 2 | Прочитать README перед commit (5 экранов, чтение) | 50 (открытие + чтение многостраничного документа) | Я не могу за тебя — это authorship-decision |
| 3 | Запушить в GitHub (`git push -u origin main`) | 0 если ты разрешишь автопуш / 10 если хочешь подтвердить | Я могу сам, нужен go |

Total owner effort: 70 (если push разрешён) / 80 (если каждый шаг подтверждается)
Critical path: №2 — без чтения README твой signoff не информирован
```

### Плохой пример (anti-pattern)

```
| # | Next action | Effort | Notes |
|---|---|---|---|
| 1 | All good | 0 | TODO |

❌ Колонка `Notes` вместо «Что агент может сделать сам»
❌ Effort=0 для непрозрачного «all good»
❌ Шкала не указана (0-100? 1-10?)
❌ TODO в значениях
```
